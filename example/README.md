# Example

This directory contains a draw.io sample and the generated outputs used to verify
text restoration, including font sizes.

## Files

- `sample.drawio`: source diagrams.net/draw.io file
- `svg/sample.svg`: sanitized intermediate SVG generated by `drawio2tikz`
- `tikz/sample.tex`: generated TikZ output
- `sample-preview.tex`: minimal LaTeX wrapper for previewing the TikZ output
- `sample-preview.pdf`: compiled PDF preview

## Regenerate

From the repository root:

```bash
uv run drawio2tikz example/sample.drawio --keep-svg --svg-dir example/svg -o example/tikz
```

Then compile the PDF preview:

```bash
cd example
lualatex -interaction=nonstopmode -halt-on-error sample-preview.tex
rm -f sample-preview.aux sample-preview.log
```

The preview should preserve the large text sizes from the draw.io labels.
